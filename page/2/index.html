然后通过sudo cat /sys/kernel/debug/tracing/trace_pipe查看信息

sudo apt-get install libbpf-dev

SEC(“tp/syscalls/sys_enter_write”)

  • tp: 表示 tracepoint，是一种在系统运行时添加触发点来捕获事件的机制。
  • syscalls: 表示系统调用，表明这个 tracepoint 用于跟踪系统调用。
  • sys_enter_write: 是具体的 tracepoint 名称，表示在进入 write 系统调用时触发。在 Linux 中，sys_enter_* tracepoints 是在系统调用进入时触发的。

pwn-出题-win-ret2dll

已merge到wiki

原理与linux下的ret2libc类似，不过需要手动提取地址，目前pwntools不能提取exe文件地址。
源代码如下

使用gcc(MinGW-W64 x86_64-ucrt-posix-seh)在windows下编译

gcc "-Wl,--disable-reloc" -g -o ret2dll.exe ret2dll.c

搭建临时环境

使用ncat
ncat -l 8080 --keep-open --exec ".\ret2dll.exe"
使用win_server
win_server ./ret2dll.exe 8080
